# Marketing & AI Discovery Strategy: Setu

To ensure "Setu" reaches the Indian Diaspora effectively and is easily "discovered" by AI tools, we need a technical foundation that is content-aware.

## 🤖 AI Discovery (The "LLM-Ready" Web)
Search has changed. People now ask AI for recommendations.
1.  **JSON-LD / Schema.org:** Implement deep structured data.
2.  **Semantic HTML:** Using `<article>`, `<section>`, and `<header>` strictly.
3.  **High-Quality Descriptions:** Crafting metadata for LLM context.

## 📢 Multi-Platform Marketing Foundation
The site will be a "Content Hub" rather than just a brochure.

### Phase 1 (Frontend Readiness)
*   **OpenGraph (OG) & Twitter Cards:** Every page will have unique images and snippets for Social Media/WhatsApp.
*   **Asset Management:** Storing high-res logo/assets in the code for social previews.

### Phase 2 (The "Marketing Robot" Backend)
Using **Next.js**, we can build **API Routes** that act as marketing assistants:
1.  **Snippet Generator:** Automatically generate IG, LinkedIn, and WhatsApp versions of blog posts.
2.  **Cross-Posting:** Ping external tools like Buffer/Make.com via Webhooks when content is published.
